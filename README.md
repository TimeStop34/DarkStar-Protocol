> Децентрализованный, скрытый P2P-протокол нового поколения

## 🚨 Проблемы

| Проблема | Решение DarkStar |
|----------|------------------|
| 🔒 Цензура и блокировки | Reality DPNS – маскировка под обычный HTTPS, неотличимость от легитимного трафика |
| 🕵️ Отсутствие анонимности | DPNCCP – многохоповая маршрутизация с контролем хопов/пинга, сквозное шифрование DPNLC |
| 🏛️ Централизованные DNS и CA | DPNNS + блокчейны – децентрализованная система имён и привязка ключей к адресам |
| 🌐 Несовместимость с IPv6 | IPvDPN – криптографические адреса, являющиеся валидными IPv6-адресами |
| 📱 Сложность настройки | Подключение по ссылке `dpn://` (IP, порт, публичный ключ) |

## 🧱 Компоненты

| Компонент | Назначение |
|-----------|------------|
| **Reality DPNS** | TLS 1.3 с подстановкой SNI легитимного домена, защита от активного зондирования |
| **DPNCCP** | Маршрутизация, проброс портов, выбор маршрута по `max_hop` и `max_ping` |
| **DPNLC** | Сквозное шифрование, аутентификация через сертификаты, SNI |
| **IPvDPN** | 128-битные адреса из публичного ключа, префикс `fd44:6b53:744e::/48` |
| **DPNNS** | Инвертированный DNS (`TLD.SLD`), блокчейн-записи имён |
| **DSChain / DPNNS Chain** | Блокчейны привязки `IPvDPN → ключи/IP` и доменов (PoW, низкая сложность) |
